# OrbitFlow

OrbitFlow é um protótipo Android criado para a Global Solution FIAP 2026.1, com o tema Indústria Espacial.

## Descrição

O OrbitFlow é uma plataforma mobile para monitoramento de lixo espacial, objetos orbitais, zonas de risco, rotas seguras e solicitações de limpeza ou desvio orbital. A proposta é apoiar empresas, agências e operadores espaciais na tomada de decisão para reduzir riscos de colisão em órbita.

## Objetivo do app

Criar uma experiência funcional em Android com dados mockados e interações reais em tela, simulando um painel de decisão para operações espaciais.

## Tecnologias usadas

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Dados mockados locais

## Funcionalidades

- Tela inicial com identidade visual espacial feita em Compose.
- Dashboard com cards de resumo, alertas recentes e atualização simulada de dados.
- Lista de objetos orbitais com filtro por risco: Todos, Baixo, Médio, Alto e Crítico.
- Lista de zonas de risco com solicitação simulada de limpeza orbital.
- Navegação entre Home, Dashboard, Objetos Orbitais e Zonas de Risco.

## Fluxo de telas

1. Home: apresenta o OrbitFlow e dá acesso às principais áreas.
2. Dashboard: resume objetos monitorados, riscos críticos, zonas ativas e solicitações abertas.
3. Objetos Orbitais: lista objetos, órbitas, riscos, distâncias e recomendações.
4. Zonas de Risco: lista regiões críticas e permite registrar uma solicitação simulada.

## Como executar no Android Studio

1. Abra a pasta do projeto `OrbitFlow` no Android Studio.
2. Aguarde a sincronização do Gradle.
3. Selecione um emulador ou dispositivo físico.
4. Execute o app pelo botão Run.
